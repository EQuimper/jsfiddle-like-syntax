# jsfiddle-like-syntax theme

A syntax like the one on JsFiddle with some tweak for Atom.

https://atom.io/packages/jsfiddle-like-syntax

Work well for:

  - Babel
  - Html
  - CSS

## Recommandations
- Font-Family reccomend: *Inconsolata*
- Ui: *Native-Ui*
